# Страницы UI

## Алгоритмы

- Список групп алгоритмов.
- Внутри группы: список алгоритмов.
- Фильтр по книге/части (через источник).

## Импорт алгоритмов из GPT

- Экран вставки JSON-ответа.
- Предпросмотр найденных алгоритмов.
- Ручное назначение группы для каждого алгоритма.
- Опциональная кнопка "Создать группу".

## Детальная страница алгоритма

- Краткое описание, когда применять, сложность.
- Инварианты, шаги, corner cases.
- Блок кода (псевдокод или язык) с копированием.
- Кнопка тренировки (может быть скрыта до следующей версии).

## Группы алгоритмов

- Детальная страница группы.
- Заголовок, описание, заметки.
- Список алгоритмов в группе.

## Термины

- Сессия чтения — процесс записи заметок + кода.
- Часть — сохраненный результат сессии чтения (как сейчас).
- Повторение — scheduled item для части.
- Группа алгоритмов — "D-куча", "КЧ-дерево", "Дейкстра" (контейнер).
- Алгоритм — операция/процедура внутри группы ("heapify", "insert").
